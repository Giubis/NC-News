GET /api/articles

Retrieves an array of all articles. Results are sorted by the 'created_at' column in descending order by default.

Queries

None

Example Request

{}

Example Response

{
  "articles": [
    {
      "title": "Seafood substitutions are increasing",
      "topic": "cooking",
      "author": "weegembump",
      "created_at": "2018-05-30T15:59:13.341Z",
      "votes": 0,
      "comment_count": 6
    }
  ]
}

POST /api/articles/:article_id/comments

Adds a new comment to the specified article by article_id.

Queries

None

Request Body

Field Description
username The username of the comment's author
body The content of the comment

Example Request

{
  "username": "butter_bridge",
  "body": "This is task 7!"
}

Example Response

{
  "comment": {
    "comment_id": 23,
    "article_id": 3,
    "body": "This is task 7!",
    "votes": 0,
    "author": "butter_bridge",
    "created_at": "2025-06-06T12:45:00.000Z"
  }
}

PATCH /api/articles/:article_id

Updates the vote count of the specified article by article_id.

Queries

None

Request Body

Field Description
inc_votes An integer indicating how much to increment or decrement the article's current vote count

Example Request

{
  "inc_votes": 10
}

Example Response

{
  "article": {
    "article_id": 1,
    "title": "Living in the shadow of a great man",
    "topic": "mitch",
    "author": "butter_bridge",
    "body": "I find this existence challenging",
    "created_at": "2020-07-09 21:11:00",
    "votes": 110,
    "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700"
  }
}
